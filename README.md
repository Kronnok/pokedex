# Pokédex Moderna

Aplicación web responsive para explorar Pokémon usando la PokéAPI.

## Archivos
- index.html
- styles.css
- app.js

## Despliegue en GitHub Pages
Esta app es estática, así que se puede publicar directamente en GitHub Pages sin construir nada.

1. Crear un repositorio en GitHub y subir estos archivos.
2. Ir a Settings > Pages.
3. En Source, elegir GitHub Actions.
4. Confirmar y esperar a que el workflow publique la web.
5. La URL quedará disponible en la sección Pages del repositorio.

Si quieres que se publique automáticamente, el proyecto ya incluye un workflow listo para hacerlo.

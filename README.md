# Pokédex Moderna

Aplicación web responsive para explorar Pokémon usando la PokéAPI.

## Archivos
- index.html
- styles.css
- app.js

## Despliegue en GitHub Pages
1. Crear un repositorio en GitHub.
2. Subir estos archivos.
3. Ir a Settings > Pages.
4. Seleccionar la rama principal como fuente.
5. Guardar y esperar a que GitHub genere la URL pública.
